## 📋 Deskripsi

**screnshoot** adalah tools Python untuk **screenshot** dan **rekam layar** dengan mudah. Dilengkapi dengan antarmuka sederhana, timer, dan pilihan area.

**Cocok untuk:**
- Mengambil screenshot cepat
- Merekam tutorial/aktivitas layar
- Otomatisasi dokumentasi visual
- Belajar Python multimedia

## 🛠️ Instalasi

### 1. Clone Repository

```bash
git clone https://github.com/hidayat-tanjung/screnshoot.git
cd screnshoot
```
### 2 Install Dependensi
```bash
pip install -r requirements.txt
```
### 3 Jalankan Tools 
```bash
python screnshoot.py
```
